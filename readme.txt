Wordpress Support Ticket System Plugin Redefined. The Best Wordpress Customer Support / Helpdesk Plugin with Robust Features & Intuitive Interface.

== Description ==
WSDesk is a Simple and User-friendly Wordpress Support Ticket System Plugin which would help you in managing your customers’ queries seamlessly. With WSDesk, all your customers’ complaints turn into trackable support tickets, making your job effortless and swift.

= Best WordPress Customer Support / HELPDESK PLUGIN =

With this help desk plugin, you can save your time and effort in managing customer issues. WSDesk is equipped with everything you need: unlimited tickets, multiple support agents, customizable email reply, complete ticket history, file attachment to tickets and lot more. 

= Why choose WSDesk? =

Offering excellent customer support is critical for the success of any organization. WSDesk has been designed keeping into consideration every issue we have faced in the process of finding the right support system for our website. Apart from being very easy to use, this plugin offers all essential feature required for a helpdesk system. It is light and Ajax based, thus it won’t interfere with your website’s loading speed. 

And moreover, it’s free!

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Thats it! you can now configure the plugin.

== Frequently Asked Questions ==

= Q. How to contact the support of WSDesk? =
A. You can get support by posting on the support section of this plugin on WordPress plugins directory or directly mailing to support@wsdesk.com

= Q. Can I use my existing WordPress theme? =
A. Sure, you can use your existing WordPress theme with WSDesk. We will just use a page for displaying support panel.

= Q. Where can I report a bug? =
A. Please let us know by posting on the support section of this plugin on WordPress plugins directory or directly mailing to support@wsdesk.com

= Q. Is WSDesk free? =
A. Yes, WSDesk basic version is free.
